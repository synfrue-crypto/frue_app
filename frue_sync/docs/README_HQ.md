# README HQ

Dieses Paket ist die Arbeitsgrundlage für Karl + Team:
- **system/** Regeln, Design, Navigation
- **briefings/** konkrete Arbeitspakete
- **contracts/** Datenverträge
- **configs/** laufzeitnahe Konfiguration
- **docs/** Dokumentation
- **checkpoints/** Wiederherstellungspunkte

Ablauf:
1) `system/Arbeitsablauf_Karl_HQ_v1.md` lesen
2) Briefings der Reihe nach umsetzen
3) Änderungen in `logs/` protokollieren
