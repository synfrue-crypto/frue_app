# Designrichtlinie Core (Kurz)

- Ton: warm, freundlich, ruhig. Weißraum bewusst.
- Typo: klare Hierarchie (H1/H2/H3, Body), ausreichend Zeilenhöhe.
- Abstände: 8px-Raster (8/16/24/32).
- Komponenten:
  - Chips: pill-förmig, Label in Groß-/Kleinschreibung.
  - Badges: bio/demeter im Produktbild rechts unten, gut sichtbar.
  - Karten: Schlagschatten sparsam, Border-Radius mittel.
- Responsiv:
  - XS: 1 Spalte · SM: 2 · MD: 3 · LG: 4+
